Terminologie Médicale

C'est la science qui étudie les termes (les mots) médicaux.

1. Préfixe :

Est ce placé avant le mot (radical) pour en modifier le sens.

  • Ex. : in (apte : inapte)
  • dé, dés (agréable : désagréable) (contraire)
  • re, ré : (répétition) faire : refaire, essayer : réessayer
  • sous, super : (infériorité/supériorité) directeur : sous-directeur, marché : supermarché

2. Suffixe :

Est ce placé après le mot (radical) pour en modifier le sens.

  • Ex. : -ment, -aison (nominalisation) hurler : hurlement, livrer : livraison
  • -ier, -ien, -iste (métier) jardin : jardinier, diriger : directeur, journal : journaliste

3. Le radical :

C'est la base, l'origine du mot.

A :

  • A, An : absence - Aphonie → absence de la voix (غياب الصوت).
  • ANA : contraire de - Anaphylaxie : état inverse de l'immunité.
  • Alg(o) : douleur - Névralgie : douleur du nerf (ألم الأعصاب).
  • All : autre - Allergie : sensibilité anormale à certaines substances.
  • Ambul : se déplacer, se promener - Somnambul : la personne qui marche en dormant.
  • Angio : vaisseaux - Angiographie : examen radiographique des vaisseaux.
  • Anti : contre - Antibiotique : médicament contre les microbes.
  • Auto(s) : de soi-même - Autogreffe : greffe prise du corps de la même personne.
  • Arthr(o) : articulation - Arthrite : inflammation de l'articulation.
  • Audi(o) : audition - Audiogramme : enregistrement de l'audition.

B :

  • Bi, Bis, Bin : deux fois - Bilatéral : (بالجانبين).
  • Bio : en rapport avec la vie - Biologie.
  • Brady : lent - Bradycardie : lenteur du battement du cœur.
  • Brachy : court - Brachyphalangie : anormalement court des doigts ou orteils.
  • Bucco : la bouche - Cavité buccale : تجويف الفم.

C :

  • Calci(o) : calcium - Calcémie : Taux de calcium dans le sang (نسبة الكالسيوم في الدم).
  • Cardi(o) : cœur - Cardiopathie : maladie du cœur.
  • Centèse : piquer, ponctionner, prélever - Rachicentèse : la ponction du LCR de la colonne vertébrale.
  • Céphal(o) : la tête - Céphalée : douleur de la tête.
  • Chlor(o) : vert - Chlorophylle : (اليخضور).
  • Cholong(i)(o) : Cholangiographie : examen radiologique des voies biliaires.
  • Chrom-, Chromat(o) : couleur, teinte - Chromosome : (الكروموسوم).
  • Cide : tuer - Les bactéricide.
  • Cran(o) : le crâne - Cranoplastie : chirurgie esthétique du crâne.
  • Cyan : la couleur bleue - Doit cyaniser : le froid.
  • Cyt(o) : la cellule - Cytologie.
  • Cyst(o) : la vessie, مثانة - Cystite : inflammation de la vessie.
